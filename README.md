CineMax – Sistema Web de Reservas de Cine

Descripción del Proyecto
CineMax es un sistema web de reservas de cine desarrollado con HTML, CSS y JavaScript. Permite a los usuarios iniciar sesión, visualizar la cartelera de películas, seleccionar asientos y registrar reservas utilizando localStorage para la gestión de datos y sesiones.

Funcionalidades
- Inicio y cierre de sesión
- Validación de sesión
- Visualización de películas
- Selección de asientos
- Registro y visualización de reservas

Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript
- localStorage
- Git y GitHub

Instrucciones de Uso
1. Abrir login.html
2. Iniciar sesión
3. Seleccionar una película
4. Elegir un asiento
5. Confirmar la reserva
6. Ver la reserva en la sección Reservas
